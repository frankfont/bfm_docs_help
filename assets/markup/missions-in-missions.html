Missions in Missions

There is no limit to the level of detail you may wish to capture using this tool. However, having too much detail in one mission diagram will reduce its utility as a coordination and communication tool. For this reason, sometimes you may want to reference detail which is modeled in a separate mission diagram.

You can link any workitem to a mission diagram that shows more detail.

How to Link a Mission to a Workitem

  1. Open the edit dialog for the workitem

  2. Add the URL of the other mission diagram

  3. Save the changes.

A clickable icon will appear for that workitem. Clicking the icon will open the linked mission in a new tab.
